================================================================================
    REQUIRED AZURE RESOURCES - Create These One by One
================================================================================

This file lists ONLY the required Azure resources needed to run the voice agent.
Optional resources (like Cosmos DB, Redis, Blob Storage) are NOT included.

Create these in your Azure subscription in the order listed below.

================================================================================
RESOURCE 1: AZURE OPENAI SERVICE
================================================================================

Purpose: Provides the LLM (Language Model) for conversation intelligence

Steps to Create:
1. Go to Azure Portal > Create a resource > Search "Azure OpenAI"
2. Click "Create"
3. Fill in details:
   - Subscription: Select your subscription
   - Resource Group: Create new or use existing (e.g., "rg-voice-agent")
   - Region: Choose region (e.g., East US, West Europe)
   - Name: Enter unique name (e.g., "aoai-voice-agent-001")
   - Pricing Tier: Standard S0
4. Click "Review + Create" then "Create"
5. Wait for deployment to complete

After Creation:
1. Go to the resource
2. Click "Keys and Endpoint" on left menu
3. Copy and save:
   - Endpoint URL (e.g., https://your-aoai.openai.azure.com)
   - Key 1 (API Key)

Deploy the Model:
1. In the Azure OpenAI resource, click "Model deployments" > "Manage Deployments"
2. This opens Azure OpenAI Studio
3. Click "Deployments" > "Create new deployment"
4. Select model: gpt-4 (version: 2024-11-20 or gpt-4o-mini)
5. Deployment name: gpt-4-1-mini (or your preferred name)
6. Click "Create"
7. Save the deployment name

What You Need for .env:
   AZURE_OPENAI_ENDPOINT=<your-endpoint-url>
   AZURE_OPENAI_KEY=<your-api-key>
   AZURE_OPENAI_DEPLOYMENT=<your-deployment-name>
   AZURE_OPENAI_API_VERSION=2024-12-01-preview
   AZURE_OPENAI_CHAT_DEPLOYMENT_ID=<your-deployment-name>
   AZURE_OPENAI_CHAT_DEPLOYMENT_VERSION=2024-11-20

================================================================================
RESOURCE 2: AZURE SPEECH SERVICES (COGNITIVE SERVICES)
================================================================================

Purpose: Provides Speech-to-Text (STT) and Text-to-Speech (TTS) capabilities

Steps to Create:
1. Go to Azure Portal > Create a resource > Search "Speech"
2. Click "Speech" (Cognitive Services)
3. Click "Create"
4. Fill in details:
   - Subscription: Select your subscription
   - Resource Group: Use same as above (e.g., "rg-voice-agent")
   - Region: Choose region (e.g., East US)
   - Name: Enter unique name (e.g., "speech-voice-agent-001")
   - Pricing Tier: Start with Free F0 (for testing) or Standard S0
5. Click "Review + Create" then "Create"
6. Wait for deployment to complete

After Creation:
1. Go to the resource
2. Click "Keys and Endpoint" on left menu
3. Copy and save:
   - Location/Region (e.g., eastus)
   - Key 1 (API Key)

What You Need for .env:
   AZURE_SPEECH_REGION=<your-region>
   AZURE_SPEECH_KEY=<your-api-key>

================================================================================
RESOURCE 3: AZURE COMMUNICATION SERVICES (ACS)
================================================================================

Purpose: Provides telephony and voice communication infrastructure

Steps to Create:
1. Go to Azure Portal > Create a resource > Search "Communication Services"
2. Click "Communication Services"
3. Click "Create"
4. Fill in details:
   - Subscription: Select your subscription
   - Resource Group: Use same as above (e.g., "rg-voice-agent")
   - Resource Name: Enter unique name (e.g., "acs-voice-agent-001")
   - Data Location: Choose location (e.g., United States)
5. Click "Review + Create" then "Create"
6. Wait for deployment to complete

After Creation - Get Connection String:
1. Go to the resource
2. Click "Keys" on left menu
3. Copy and save:
   - Connection String (Primary)
   - Endpoint URL

After Creation - Provision a Phone Number (for PSTN calling):
1. In the ACS resource, click "Phone numbers" on left menu
2. Click "Get" button
3. Select:
   - Country/region: Your country (e.g., United States)
   - Number type: Toll-free or Geographic
   - Capabilities: Check "Inbound calls" and "Outbound calls"
4. Search for available numbers
5. Select a number and click "Buy"
6. Copy the phone number (format: +1XXXXXXXXXX)

Note: Phone number costs money (~$1-2/month + per-minute usage fees)
Note: You can test without phone number using browser-only mode

What You Need for .env:
   ACS_CONNECTION_STRING=<your-connection-string>
   ACS_ENDPOINT=<your-endpoint-url>
   ACS_SOURCE_PHONE_NUMBER=<your-phone-number>

================================================================================
SUMMARY - What to Add to Your .env File
================================================================================

After creating all resources above, create a .env file in the root folder:
c:\Hackathon\art-voice-agent-accelerator\.env

Copy this template and fill in your values:

# Azure OpenAI
AZURE_OPENAI_ENDPOINT=https://<your-aoai-name>.openai.azure.com
AZURE_OPENAI_KEY=<your-key-from-step-1>
AZURE_OPENAI_DEPLOYMENT=gpt-4-1-mini
AZURE_OPENAI_API_VERSION=2024-12-01-preview
AZURE_OPENAI_CHAT_DEPLOYMENT_ID=gpt-4-1-mini
AZURE_OPENAI_CHAT_DEPLOYMENT_VERSION=2024-11-20

# Azure Speech Services
AZURE_SPEECH_REGION=<your-region-from-step-2>
AZURE_SPEECH_KEY=<your-key-from-step-2>

# Azure Communication Services
ACS_CONNECTION_STRING=<your-connection-string-from-step-3>
ACS_ENDPOINT=https://<your-acs-name>.communication.azure.com
ACS_SOURCE_PHONE_NUMBER=<your-phone-number-from-step-3>

# Runtime Settings
ENVIRONMENT=dev
ACS_STREAMING_MODE=media
BASE_URL=http://localhost:8010

================================================================================
COST ESTIMATE (Monthly)
================================================================================

Azure OpenAI:
- Pay per token usage
- Approx $0.03 per 1000 tokens (varies by model)
- Testing: ~$10-30/month depending on usage

Azure Speech Services:
- Free tier: 5 hours/month free for STT, limited TTS
- Standard: $1 per hour for STT, varies for TTS
- Testing: Can use free tier

Azure Communication Services:
- Phone number: ~$1-2/month rental
- Calling: ~$0.004-0.013 per minute (varies by region)
- Testing with phone: ~$5-10/month

TOTAL for minimal testing: ~$15-50/month

================================================================================
TESTING WITHOUT PHONE (Browser Only)
================================================================================

If you want to test without provisioning a phone number:

1. Skip the phone number provisioning in Resource 3
2. In your .env file, you can skip or comment out:
   # ACS_SOURCE_PHONE_NUMBER=+1XXXXXXXXXX
3. Access the agent only through the web browser interface
4. You'll still need the ACS_CONNECTION_STRING and ACS_ENDPOINT

This reduces cost significantly (no phone rental or per-minute charges).

================================================================================
TROUBLESHOOTING
================================================================================

If you get "quota exceeded" errors:
- You may need to request quota increase for Azure OpenAI in your region
- Go to Azure OpenAI resource > Quotas > Request quota increase

If you can't find Speech Services:
- Search for "Cognitive Services" instead
- Create a "Cognitive Services" resource with Speech enabled

If phone number provisioning fails:
- Not all regions/countries support phone numbers
- Try different region or use browser-only mode
